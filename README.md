# Splendor Game (璀璨宝石)

这是一个基于经典桌游 **Splendor**（璀璨宝石）的网页版实现，包含单机和多人游戏模式。

## 🎯 项目概述

Splendor 是一款经典的策略桌游，玩家扮演文艺复兴时期的宝石商人，通过收集宝石代币来购买卡牌，卡牌提供永久宝石折扣，贵族卡提供胜利点数，最先达到15点的玩家获胜。

本项目实现了：
- 完整的游戏规则和逻辑
- 精美的网页界面
- 单机游戏模式
- 多人游戏模式（基于 WebSocket）
- 响应式设计，支持不同设备

## ✨ 功能特性

- **完整游戏规则**：实现了 Splendor 的所有核心规则
- **美观界面**：现代化的 UI 设计，流畅的动画效果
- **多人对战**：支持 2-4 人在线对战
- **本地游戏**：单机模式，无需网络连接
- **游戏配置**：根据玩家人数自动调整游戏参数
- **键盘快捷键**：支持快捷键操作，提升游戏体验
- **游戏状态管理**：实时显示游戏状态和玩家信息

## 🚀 快速开始

### 单机游戏模式

1. 直接在浏览器中打开 `index.html` 文件
2. 选择玩家人数（2-4人）
3. 点击「开始游戏」按钮
4. 按照游戏规则进行游戏

### 多人游戏模式

1. 启动服务器：
   ```bash
   cd server
   npm install
   npm start
   ```

2. 打开 `index.html` 文件
3. 选择「多人游戏」选项
4. 输入服务器地址（默认为 `localhost:8080`）
5. 创建或加入游戏房间

## 📜 游戏规则

### 组件
- **宝石卡牌**：分 Tier 1/2/3，提供宝石折扣和胜利点数
- **贵族卡**：提供大量胜利点数，需要满足特定宝石折扣要求
- **宝石代币**：6种颜色：白、蓝、绿、红、黑、金

### 目标
成为第一个获得15个或以上胜利点数的玩家。

### 游戏流程
1. 每位玩家轮流执行以下行动之一：
   - 拿取3个不同颜色的宝石
   - 拿取2个相同颜色的宝石（该颜色需至少有4个）
   - 购买一张可用的卡牌
   - 预订一张卡牌并拿取一个金色代币
2. 当有玩家达到15个胜利点数时，游戏结束，该玩家获胜。

### 行动说明

#### 拿取宝石
- **3个不同颜色**：从宝石池中拿取3个不同颜色的宝石，每种颜色各1个
- **2个相同颜色**：从宝石池中拿取2个相同颜色的宝石，但该颜色的宝石必须至少有4个
- **金色代币**：只能通过预订卡牌获得，可作为万能宝石使用

#### 购买卡牌
- 使用宝石代币购买桌面上的卡牌
- 已购卡牌提供永久宝石折扣，降低后续卡牌的购买成本
- 部分卡牌提供胜利点数

#### 预订卡牌
- 将桌面上的卡牌放入个人预留区
- 获得1个金色代币（如果还有）
- 预留的卡牌可以在后续回合购买
- 每个玩家最多可以预留3张卡牌

#### 贵族拜访
- 当玩家的宝石折扣满足贵族卡的要求时，自动获得该贵族卡
- 贵族卡提供大量胜利点数

## 📁 项目结构

```
splendor-game/
├── server/              # 多人游戏服务器
│   ├── package.json     # 服务器依赖
│   └── server.js        # WebSocket 服务器

├── index.html           # 主游戏页面
```

### 核心文件说明

- **splendor-core.js**：游戏核心逻辑，实现了完整的游戏规则

## 🎮 游戏操作

### 鼠标操作
- 点击宝石池中的宝石：选择宝石
- 点击卡牌：选择卡牌
- 点击按钮：执行相应操作

### 键盘快捷键
- `1`：选择「拿取3个不同宝石」
- `2`：选择「拿取2个相同宝石」
- `B`：购买选中的卡牌
- `R`：预订选中的卡牌
- `?`：显示游戏规则

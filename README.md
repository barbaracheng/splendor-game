# Splendor Game

这是一个类似Steam上的Splendor（璀璨宝石）的桌游游戏，包含两个版本：
1. HTML网页版本
2. 微信小游戏版本

## 游戏规则

Splendor是一款经典的策略桌游，玩家扮演文艺复兴时期的宝石商人，通过收集宝石代币来购买卡牌，卡牌提供永久宝石折扣，贵族卡提供胜利点数，最先达到15点的玩家获胜。

### 组件
- 宝石卡牌（分Tier 1/2/3）
- 贵族卡
- 宝石代币（6种颜色：白、蓝、绿、红、黑、金）

### 目标
成为第一个获得15个或以上胜利点数的玩家。

### 游戏流程
1. 每位玩家轮流执行以下行动之一：
   - 拿取3个不同颜色的宝石
   - 拿取2个相同颜色的宝石
   - 购买一张可用的卡牌
   - 预订一张卡牌并拿取一个金色代币
2. 当有玩家达到15个胜利点数时，游戏结束，该玩家获胜。
